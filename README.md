🍕 Snack & Juice Management System | نظام إدارة المطاعم الذكي
منظومة رقمية متكاملة لإدارة الطلبات، التوصيل، والمخزون باستخدام تقنيات الربط الديناميكي.

هذا المشروع ليس مجرد واجهة لطلب الطعام، بل هو نظام إدارة تشغيلي (Operational Management System) مصمم بلغة #C. يعتمد على مفهوم تجريد البيانات (Data Abstraction) عبر كلاس JsonDatabase لضمان مرونة عالية في تخزين واسترجاع الطلبات دون الحاجة لخوادم SQL معقدة.

💎 العبقرية الهندسية في هيكلية النظام (System Architecture)
ما يميز هذا النظام هو "مركزية الطلب" (Order-Centric Design)، حيث يتم الربط كالتالي:

محرك تخزين JSON (The JSON Engine): تم بناء كلاس JsonDatabase.cs ليعمل كـ ORM مصغر، حيث يقوم بتحويل الكائنات (Objects) إلى نصوص JSON وحفظها، مما يضمن استمرارية البيانات (Data Persistence) وسهولة نقل المشروع.

نظام التتبع المتعدد (Multi-Tracking System): يربط النظام بين ثلاث وحدات رئيسية في آن واحد:

وحدة التصنيف: (بيتزا، حلويات، عصائر) مع ميزة الحساب التلقائي للأسعار بناءً على الكميات.

وحدة الهوية: ربط كل طلب بـ CustomerId فريد يتم توليده تلقائياً لضمان عدم تداخل الحسابات بين الزبائن.

التحكم في حالة الطلب (Status Lifecycle): النظام يطبق منطق (State Machine) في شاشة Remote Delivery؛ حيث ينتقل الطلب برمجياً من Pending (قيد الانتظار) إلى Delivering (جاري التوصيل) ثم Delivered (تم التسليم)، مع تحديث فوري لقاعدة البيانات.

🚀 الوظائف المتقدمة والتحليلات (Advanced Functionalities)
🔐 نظام الصلاحيات والأمان (Authentication Layer)
Login Security: واجهة دخول ذكية تدعم مستويات وصول مختلفة (Admin/User).

Session Management: يتم تخصيص تجربة المستخدم بناءً على الهوية (مثل صلاحيات التحكم الكاملة للمسؤول "Admin").

🛒 محرك الطلبات الذكي (Order Management Engine)
سلة تسوق ديناميكية: معالجة الطلبات في وقتها الحقيقي (Real-time Processing) عبر كلاس الخدمات المخصص.

الفلترة الذكية: القدرة على تصفية مئات الطلبات حسب "رقم العميل" أو "حالة الطلب" في شاشة التوصيل بضغطة زر واحدة.

📊 التوصيل والعمليات (Remote Delivery Module)
لوحة تحكم العمليات: شاشة مخصصة لمراقبة تدفق الطلبات وتحديث حالتها فورياً لتنعكس في ملف البيانات المركزي، مما يسهل عملية المتابعة اللوجستية.

📂 رؤية داخل الكود (Logic Mapping)
النظام يدير تدفق البيانات بذكاء:

Model-Service Link: فصل منطق الحسابات (Business Logic) عن واجهات العرض (UI) لسهولة الصيانة.

Auto-ID Generator: نظام داخلي ذكي لتوليد أرقام العملاء والطلبات لمنع تضارب البيانات.

Data Serialization: تحويل القوائم المعقدة إلى ملف orders.json واسترجاعها بذكاء وسرعة.
