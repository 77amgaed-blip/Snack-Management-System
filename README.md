# 🍕 Snack & Juice Management System  
نظام ذكي لإدارة طلبات المطاعم والعصائر والحلويات  

## مقدمة
نظام Snack & Juice Management System هو حل برمجي متكامل يعتمد على لغة #C يتيح لك إدارة طلبات العملاء، التوصيل، المخزون، والصلاحيات بشكل سهل وفعال، مع نظام تخزين بيانات ذكي يعتمد على ملفات JSON بدون الحاجة إلى قواعد بيانات معقدة.

---

## المميزات  
- **تصميم ديناميكي** لإدارة الطلبات بين التصنيفات (بيتزا، حلويات، عصائر) مع حساب تلقائي للأسعار.
- **تتبع الطلبات** عبر حالات متعددة (بانتظار التنفيذ، جاري التوصيل، تم التسليم).
- **نظام صلاحيات** (مستخدم / مسؤول).
- **لوحة تحكم** لإدارة تدفق الطلبات ومراقبة حالة التوصيل.
- **توليد أرقام تلقائي** للعملاء والطلبات.
- **فلترة ذكية** للطلبات حسب العميل أو الحالة.
- **واجهة سهلة الاستخدام** وصديقة للمطورين وإمكانية تعديل وتجربة الكود بسهولة.

---

## بنية النظام  
- **لغة البرمجة:** #C
- **تخزين البيانات:** ملفات JSON بواسطة كلاس مخصص (JsonDatabase.cs)
- **هيكلية الكود:** فصل بين الواجهات والمنطق البرمجي (Business Logic & UI)

---

## المتطلبات  
- تثبيت .NET Framework أو .NET Core المناسب لتشغيل برامج C#
- نظام تشغيل Windows (يفضل بيئة Visual Studio)

---

## طريقة التشغيل  
1. قم باستنساخ المستودع:
   ```bash
   git clone https://github.com/77amgaed-blip/Snack-Management-System.git
   ```
2. افتح المشروع باستخدام Visual Studio أو أي محرر يدعم C#
3. تأكد من إعداد مسار ملفات البيانات (orders.json إلخ)
4. قم بالبناء (Build) ثم تشغيل النظام

---

## أمثلة الاستخدام  
- إضافة طلب جديد (بيتزا/عصير/حلوى) مع حساب تلقائي للسعر.
- متابعة حالة الطلب وتغييرها من واجهة التحكم.
- تصفية الطلبات وتوليد تقارير.
